A historic building sample demonstrating Forge Viewer and its extension.
The building components can be selected by mouse clicking and moved by mouse moving.

A sample historic building demonstrating the integration of Forge Viewer and visual programming tools including Dynamo and Grasshopper,through the use of Flux.io.

First make sure you have the URL for this page using "https" - secured site required by Flux.io.

First login to your account on Flux.io and then continue to enable the Flux App.
Then load Autodesk Viewer with the sample model
